统计笑傲江湖这本书里，任盈盈的名字总共出现了多少次

```python

with open(r"JinYong\笑傲江湖.txt",'r',encoding='gbk',errors='ignore') as text:
    txt=text.read()

import jieba.posseg as jb

words=[]
for word, flag in jb.cut(txt):
    if flag =="nr":
        words.append(word)

import pandas as pd

print(pd.Series(words).value_counts()[:40])
```
