在任意文件夹下的任意页面，可以引入DocCardList组件

```jsx title='docs/sidebar/index.md'
import DocCardList from '@theme/DocCardList';

<DocCardList />
```

:::caution

如果ceshi.md不属于任意文件夹，则无法使用DocCardList组件。

:::
