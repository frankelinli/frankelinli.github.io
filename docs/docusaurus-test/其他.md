docusaurus.config.js

```js
export default {
  // ...
  presets: [
    [
      '@docusaurus/preset-classic',
      {
        blog: {
          blogTitle: 'Docusaurus blog!',
          blogDescription: 'A Docusaurus powered blog!',
          postsPerPage: 'ALL',
        },
      },
    ],
  ],
};
```

设置blog首页显示全部博客，无限拉长：postsPerPage: 'ALL',

设置blog首页显示3篇文章：postsPerPage: '3,
